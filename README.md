# Deekas Virtual Nail Try-On

A beautiful, interactive virtual nail try-on website where users can experiment with different nail colors and shapes.

## Features

- **Homepage**: Welcoming landing page with animated logo and start button
- **Virtual Try-On**: Interactive hand display with clickable nail overlays
- **Color Selection**: 15 different nail polish colors to choose from
- **Shape Options**: Three nail shapes - Round, Square, and Almond
- **Download**: Save your nail design as a PNG image
- **Share**: Social media sharing options (visual only)
- **Responsive Design**: Works perfectly on both desktop and mobile devices

## How to Use

1. **Open the Website**: Simply open `index.html` in any modern web browser
2. **Start Try-On**: Click the "Start Try-On" button on the homepage
3. **Select Nails**: Click on individual nails to select/deselect them
4. **Choose Colors**: Click on any color circle to change the nail color
5. **Pick Shapes**: Select from Round, Square, or Almond nail shapes
6. **Download**: Click "Download Design" to save your creation
7. **Share**: Click "Share Design" to see social media options

## Technical Details

- **Frontend Only**: Pure HTML, CSS, and JavaScript - no backend required
- **SVG Graphics**: Hand image is generated using SVG for crisp, scalable graphics
- **Canvas Export**: Download functionality uses HTML5 Canvas to create PNG images
- **Responsive Design**: CSS Grid and Flexbox for mobile-friendly layout
- **Modern UI**: Beautiful pink and white theme with smooth animations
